# Loghi Vygo - file identici alla scelta

Questa cartella contiene i file PNG ricavati dalla **Tavola 1 originale - Camion e percorso** senza ridisegnare il logo.

## Scelta approvata

- App icon principale: camion su sfondo nero, riga 1 a sinistra.
- Logo orizzontale: camion + strada + scritta Vygo, riga 1 centrale.
- Icona piccola/favicon: quadratino nero con V + strada, riga 2.
- Slogan fisso: Move. Manage. Succeed.

## File

- `vygo-app-icon-truck-black-exact.png`: ritaglio originale identico.
- `vygo-app-icon-truck-black-1024.png`: versione grande per app e materiali.
- `vygo-logo-horizontal-truck-exact.png`: ritaglio originale identico.
- `vygo-logo-horizontal-truck-3200.png`: versione grande per sito/script/materiali.
- `vygo-small-icon-v-road-black-exact.png`: ritaglio originale identico.
- `vygo-small-icon-v-road-black-512.png`: versione grande per favicon/app secondaria.

Nota: questi file rispettano esattamente la grafica scelta. Per deposito marchio definitivo e uso professionale finale andra poi fatto un tracciamento vettoriale fedele, senza cambiare forme, proporzioni o stile.
