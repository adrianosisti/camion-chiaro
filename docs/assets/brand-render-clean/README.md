# Vygo brand render-clean

Questa cartella contiene i file finali derivati dal primo rendering scelto.

Regola importante: questi asset non sono reinterpretazioni creative. Sono pulizie tecniche dei tre elementi selezionati:

- icona app: camion su sfondo nero dalla tavola 1, riga 1, sinistra;
- logo orizzontale: camion + scritta Vygo dalla tavola 1, riga 1, centro;
- icona piccola: V + strada dalla tavola 1, riga 2.

## File da usare

- `vygo-app-icon-render-clean-1024.png`: icona app iOS/Android, pulita e senza cornice sporca.
- `vygo-logo-horizontal-render-raster-clean-1600.png`: logo orizzontale fedele al rendering, per sito, login e splash.
- `vygo-small-icon-render-clean-512.png`: icona piccola/favicon.
- `vygo-app-icon-render-clean.svg`, `vygo-logo-horizontal-render-clean.svg`, `vygo-small-icon-render-clean.svg`: tracciati vettoriali di riferimento.

Il rendering originale era a bassa risoluzione e conteneva bordi/cornici sporche. Per l'icona app e favicon viene usata la versione tracciata dal rendering, cosi resta pulita ad alta risoluzione. Per il logo orizzontale viene usata la versione raster ripulita, per mantenere la scritta piu fedele.
