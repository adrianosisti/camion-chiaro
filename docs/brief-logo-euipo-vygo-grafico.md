# Brief grafico - vettorializzazione marchio Vygo per EUIPO

## Obiettivo

Rifare in vettoriale professionale il logo Vygo gia in uso, senza cambiarne stile, proporzioni o identita.

Il lavoro non deve essere un redesign. Deve essere una vettorializzazione fedele, pulita e depositabile.

## File riferimento

Usare come riferimento principale:

- `docs/assets/euipo-vygo/reference-vygo-logo-current-site.png`

Per icona app:

- `docs/assets/euipo-vygo/reference-vygo-app-icon-current.png`

SVG provvisori disponibili solo come base di lavoro:

- `docs/assets/euipo-vygo/draft-trace-vygo-logo-current-site.svg`
- `docs/assets/euipo-vygo/draft-trace-vygo-app-icon-current.svg`

## Cosa deve consegnare il grafico

1. Logo principale Vygo orizzontale in formato `.ai`, `.svg`, `.pdf`.
2. Logo principale senza payoff.
3. Logo completo con payoff `Move. Manage. Succeed.`
4. Versione monocromatica nera.
5. Versione monocromatica bianca.
6. Icona/simbolo app separato.
7. File con testo convertito in tracciati.
8. Palette colori ufficiale con codici HEX, RGB, CMYK.
9. Mini manuale d'uso: area di rispetto, dimensione minima, uso su fondo chiaro/scuro.

## Requisiti tecnici

- Nessun bordo sporco.
- Nessun ritaglio visibile.
- Nessun pixel o immagine raster incorporata nel file vettoriale finale.
- Testi convertiti in tracciati.
- Proporzioni uguali al logo attuale.
- Colori coerenti con il logo esistente.
- File pronto per deposito marchio e uso commerciale.

## Nota per EUIPO

Per il primo deposito si consiglia di valutare il logo senza payoff, cosi il marchio Vygo resta piu libero e forte. Payoff e icona possono essere protetti con depositi separati se il consulente marchi lo ritiene utile.
