# Vygo - pacchetto contrattuale

Questa cartella contiene bozze operative per preparare i contratti Vygo.

Sono documenti di lavoro, non pareri legali definitivi. Prima dell'uso commerciale vanno verificati da avvocato, consulente privacy e commercialista.

## Documenti

- `01-contratto-saas-b2b-vygo.md` - contratto principale tra Vygo e azienda cliente.
- `02-condizioni-generali-vygo.md` - condizioni generali del servizio.
- `03-dpa-vygo.md` - accordo trattamento dati / nomina responsabile trattamento.
- `04-ordine-commerciale-vygo.md` - scheda economica/piano acquistato.
- `05-accordo-pilota-vygo.md` - accordo per test gratuito o agevolato.
- `06-checklist-legale-vygo.md` - cosa far controllare prima del lancio.

## Struttura consigliata per clienti paganti

1. Ordine commerciale firmato.
2. Contratto SaaS B2B.
3. Condizioni generali.
4. DPA privacy.
5. Informative per dipendenti/autisti accettate in app.

## Strategia

Per i primi test usare l'accordo pilota.
Per i clienti veri usare contratto minimo 12 mesi, setup iniziale non rimborsabile e penale moderata in caso di recesso anticipato.
