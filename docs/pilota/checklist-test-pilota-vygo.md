# Vygo - Checklist test pilota

Versione: 2026-07-03

Usare questa checklist durante la prova con azienda pilota. Ogni voce deve essere provata su dati reali o quasi reali.

Legenda esito:

- OK: funziona bene;
- ATTENZIONE: funziona ma va migliorato;
- BLOCCANTE: impedisce l'uso reale;
- NON TESTATO: da provare.
